SOUNDRAIN – Aplicación para música descargada

He decidido desarrollar una aplicación similar a SoundCloud, que permite a los usuarios reproducir y organizar sus canciones descargadas. La app cuenta con funcionalidades pensadas para mejorar la interacción y la comunicación con el usuario, ofreciendo una experiencia más intuitiva y fluida.

Toda la interacción principal se realiza desde una página inicial, en la cual se muestran de manera inmediata todas las canciones disponibles en el dispositivo del usuario. Para el diseño visual, opté por una paleta de colores compuesta por tonos negros y naranjas, con el objetivo de mantener un estilo sencillo, coherente y agradable a la vista a lo largo de toda la aplicación.

Además, la aplicación incorpora dos fragments específicos: uno dedicado a la sección de “Explorar”, cuya función principal es permitir al usuario localizar canciones disponibles, y otro dedicado a la “Biblioteca”, en la que se visualizan todas las canciones almacenadas localmente. Esta estructura facilita la navegación y organiza la información de manera clara.

Para indicar el progreso de reproducción de cada canción seleccionada, utilicé el componente Progress Indicator, que fue elegido previamente en la presentación de componentes. Este elemento permite al usuario tener un control visual y directo sobre el avance de la canción en reproducción, mejorando la experiencia general dentro de la app.
